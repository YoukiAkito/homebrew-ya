# YoukiAkito YA

## How do I install these formulae?

`brew install youkiakito/ya/<formula>`

Or `brew tap youkiakito/ya` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "youkiakito/ya"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
